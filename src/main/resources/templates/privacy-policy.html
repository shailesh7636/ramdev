Privacy Policy

📋 Overview

This Privacy Policy explains how Jai Ramdev Agriculture ("we", "our", or "the app") collects, uses, and protects your information when you use our mobile application. This app is a video portal for agricultural thresher machine content, operated by Mukeshbhai, Halwad, Morbi, Gujarat, India.

By using this app, you agree to the collection and use of information as described in this policy.

📥 Information We Collect

We collect only the information necessary to provide our service:

  • Full Name — provided during account registration
  • Mobile Number — used as your unique login identifier
  • Password — stored securely using BCrypt hashing (never stored in plain text)
  • Account creation date — recorded automatically when you register
No Email Required No Location Data No Camera Access No Contacts Access

⚙️ How We Use Your Information

Your information is used solely for the following purposes:

  • To authenticate you and manage your account securely
  • To provide access to agricultural video content
  • To maintain session security using JWT (JSON Web Tokens)
  • To allow administrators to manage user accounts
We do not use your data for advertising, profiling, or any commercial purpose.

🎬 Video Content

All videos displayed in this app are uploaded and managed by authorized administrators only. Regular users cannot upload content. Videos are stored on Cloudinary's cloud infrastructure and served securely to authenticated users.

  • Video titles, descriptions, categories, and thumbnails are managed by admins
  • Users can only view videos — no user-generated content is accepted

🔒 Data Storage & Security

We take the security of your data seriously:

  • Account data is stored in a secure PostgreSQL database hosted on Supabase
  • Passwords are encrypted using BCrypt — never stored in plain text
  • Sessions are managed using JWT tokens with expiry controls
  • The backend is hosted on Render with HTTPS enforced
  • Media files are stored on Cloudinary with secure delivery
